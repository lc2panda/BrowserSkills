## Installation Process Summary

### Step 1: MCP Server Setup (✅ COMPLETE)
```bash
git clone https://github.com/lc2panda/mcp.git
cd mcp
npm install
npm run build
node dist/index.js  # Starts MCP server on port 8765
```

### Step 2: Chrome Extension Setup (🔧 MANUAL)
1. Open Chrome browser
2. Visit Chrome Web Store
3. Search for "Browser MCP"
4. Click "Add to Chrome"
5. Pin extension and click "Connect"

### Step 3: AI Application Configuration (🔧 MANUAL)
Add to your AI app's MCP configuration:
```json
{
  "mcpServers": {
    "browsermcp": {
      "command": "node",

    }
  }
}
```
